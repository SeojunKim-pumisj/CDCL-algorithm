# CDCL Algorithm Test Examples


### 예시 1: 만족 가능 (Satisfiable)

-   **논리식**: (x₁ ∨ ¬x₂) ∧ (¬x₁ ∨ x₂)
-   **입력**:
    ```
    1 -2
    -1 2
    ```
-   **예상 결과**: 만족 가능. 가능한 해: {1: True, 2: True}

---

### 예시 2: 만족 가능 (Satisfiable)

-   **논리식**: (x₁ ∨ x₂)
-   **입력**:
    ```
    1 2
    ```
-   **예상 결과**: 만족 가능. 가능한 해: {1: True, 2: False}

---

### 예시 3: 만족 불가능 (Unsatisfiable)

-   **논리식**: (x₁) ∧ (¬x₁)
-   **입력**:
    ```
    1
    -1
    ```
-   **예상 결과**: 만족 불가능.

---

### 예시 4: 만족 불가능 (Unsatisfiable)

-   **논리식**: (x₁ ∨ x₂) ∧ (x₁ ∨ ¬x₂) ∧ (¬x₁ ∨ x₂) ∧ (¬x₁ ∨ ¬x₂)
-   **입력**:
    ```
    1 2
    1 -2
    -1 2
    -1 -2
    ```
-   **예상 결과**: 만족 불가능.

---

### 예시 5: 만족 가능 (Satisfiable)

-   **논리식**: (¬x₁ ∨ x₂) ∧ (¬x₂ ∨ x₃) ∧ (x₁ ∨ ¬x₃)
-   **입력**:
    ```
    -1 2
    -2 3
    1 -3
    ```
-   **예상 결과**: 만족 가능. 가능한 해: {1: True, 2: True, 3: True}

---

### 예시 6: 만족 불가능 (Unsatisfiable)

-   **논리식**: (x₁ ∨ x₂) ∧ (x₃ ∨ x₄) ∧ (x₅ ∨ x₆) ∧ (¬x₁ ∨ ¬x₃) ∧ (¬x₁ ∨ ¬x₅) ∧ (¬x₃ ∨ ¬x₅) ∧ (¬x₂ ∨ ¬x₄) ∧ (¬x₂ ∨ ¬x₆) ∧ (¬x₄ ∨ ¬x₆)
-   **입력**:
    ```
    1 2
    3 4
    5 6
    -1 -3
    -1 -5
    -3 -5
    -2 -4
    -2 -6
    -4 -6
    ```
-   **예상 결과**: 만족 불가능.

---

### 예시 7: 만족 가능 (Satisfiable)

-   **논리식**: (x₁ ∨ x₂ ∨ x₃) ∧ (¬x₁ ∨ ¬x₂) ∧ (¬x₂ ∨ ¬x₃) ∧ (¬x₁ ∨ ¬x₃) ∧ (x₄ ∨ x₅) ∧ (¬x₄ ∨ ¬x₅) ∧ (¬x₁ ∨ ¬x₄)
-   **입력**:
    ```
    1 2 3
    -1 -2
    -2 -3
    -1 -3
    4 5
    -4 -5
    -1 -4
    ```
-   **예상 결과**: 만족 가능. 가능한 해: {1: True, 2: False, 3: False, 4: False, 5: True}
